# drone-gen
Код генератора для летающей лаборатории

## Подключение
Параметры последовательного порта 115200 8N1

# Протокол

Смотри файл [protocol.html](protocol.html)

```bash
 # from 100 000 Hz to 400 000 000 Hz
 gen.set_freq 110000000
 # from 1 to 511
 gen.set_pow 511

```
